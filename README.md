# AI-ML-internship-tasks (Task 1)
**Task Objectives:**
The objective of this task is to learn how to:
1) Load a dataset using Python (pandas / seaborn)
2) Understand dataset structure and statistics
3) Perform basic exploratory data analysis (EDA)
4) Visualize data distributions and relationships using graphs

**Iris Dataset**

The Iris dataset is a classic dataset in machine learning used for classification and data analysis tasks.
**Features:**
Sepal Length
Sepal Width
Petal Length
Petal Width
Species (Setosa, Versicolor, Virginica)

**Models Applied**
This task focuses only on data exploration and visualization.

**Key Results and Findings**
Dataset is clean with no missing values.
Clear separation observed between Iris species in feature space.
